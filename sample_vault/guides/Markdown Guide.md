# Markdown 記法ガイド

[[index|← ホーム]] へ戻る。

## 見出し

```
# H1
## H2
### H3
```

## 強調

- **太字** は `**text**`
- *斜体* は `*text*`
- ~~打ち消し~~ は `~~text~~`
- ==ハイライト== は `==text==`（Obsidian独自）

## 引用

> 通常の引用はこのように表示されます。
> 複数行も OK です。

## チェックリスト

- [x] wikilink対応
- [x] callout対応
- [x] シンタックスハイライト
- [ ] 数式サポート（将来対応予定）

## コールアウト一覧

> [!NOTE] NOTE
> 情報を提供するときに使います。

> [!ABSTRACT] ABSTRACT / SUMMARY / TLDR
> 要約を書くときに使います。

> [!EXAMPLE] EXAMPLE
> コード例や具体例を示すときに使います。

> [!QUOTE] QUOTE
> 引用文に使います。
